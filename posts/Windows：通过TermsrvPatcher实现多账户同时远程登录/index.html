Windows：通过TermsrvPatcher实现多账户同时远程登录

2025年08月17日

TermsrvPatcher 是一个在 Github 上开源的项目，通过该项目可以实现非服务器版 Windows 系统多账户同时远程登录。

Github

项目名称：TermsrvPatcher
项目地址：https://github.com/fabianosrc/TermsrvPatcher
项目作者：fabianosrc
开源协议：MIT

保存以下代码为termsrvPatcher.ps1文件，使用“PowerShell（管理员）”运行，重启计算机即可。(支持Windows 11 24H2系统)
